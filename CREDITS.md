# CREDITS

Third-party assets used by SF Sky Courier, with exact sources and licenses.

## Kenney — "Mini Characters" (v1.0, released 2024-07-17)

- **What:** 12 rigged + animated low-poly humanoid characters
  (`character-male-a…f`, `character-female-a…f`, 723 triangles each,
  32 shared animations: idle/walk/sit/emotes/holding/…), plus small
  accessory props (glasses, sunglasses, canes, wheelchairs) and the
  shared `colormap.png` palette texture.
- **Where in repo:** `assets/npcs/glb/` (GLB format only; the zip's FBX/OBJ
  duplicates and preview images were not kept). Pack license kept verbatim
  at `assets/npcs/License.txt`.
- **Downloaded:** 2026-07-16 via curl from the official kenney.nl page
  <https://kenney.nl/assets/mini-characters>
  (file `kenney_mini-characters.zip`, 2.4 MB).
- **License:** **CC0 1.0** (Creative Commons Zero, public domain) —
  <https://creativecommons.org/publicdomain/zero/1.0/>.
  No attribution required; credited here anyway. Support: kenney.nl/donate.

## Everything else

All other models (player vehicles, bridges, trees, NPC props like
laptops/surfboards/dogs/bikes) are original primitive-based meshes built
in code in this repo, or come from the map data sources credited on the
in-game attribution line (OpenStreetMap, OpenFreeMap, ESRI World Imagery,
Mapzen/AWS terrain tiles).
